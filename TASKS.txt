# Completed MVP Features

- [x] Phaser.js game setup
- [x] Player character (green square)
- [x] Horizontal movement (left/right with arrows or A/D)
- [x] Shooting projectiles (Space or W)
- [x] Enemy spawning (red squares falling from top)
- [x] Collision detection (bullets hit enemies, enemies hit player)
- [x] Score system (+10 for kills, -5 for damage)
- [x] Game over state (when score < -50)
- [x] Placeholder graphics (no external assets)
